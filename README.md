# 📊 Imersão Dados Alura

Projeto desenvolvido durante a **Imersão Dados da Alura**, com foco em **análise exploratória de dados (EDA)**, **Data Cleaning** e **visualização interativa**, utilizando Python e bibliotecas amplamente usadas na área de Ciência de Dados.

## 🎯 Objetivo do Projeto

O objetivo deste projeto é analisar um conjunto de dados reais, aplicando etapas fundamentais de **limpeza, tratamento e exploração dos dados**, a fim de extrair insights relevantes e apresentar os resultados de forma visual e interativa.

## 🧹 Data Cleaning

Nesta etapa, foram realizadas as seguintes ações:

- Tratamento de valores nulos e inconsistentes  
- Padronização dos nomes das colunas  
- Conversão de tipos de dados  
- Filtragem e organização das informações  
- Preparação dos dados para análise exploratória  

## 📈 Análise Exploratória de Dados (EDA)

Após a limpeza dos dados, foram realizadas análises exploratórias, incluindo:

- Estatísticas descritivas  
- Agrupamentos e comparações entre categorias  
- Identificação de padrões e tendências  
- Criação de gráficos para melhor interpretação dos dados  

## 🖥️ Aplicação Interativa

O projeto conta com uma aplicação desenvolvida em **Streamlit**, que permite ao usuário:

- Filtrar os dados por ano  
- Explorar informações por categorias  
- Visualizar gráficos e métricas de forma dinâmica

Aplicação: https://projeto-dados-alura.streamlit.app/

## 🛠️ Tecnologias Utilizadas

- Python  
- Pandas  
- Streamlit  
- Plotly  
- PyCountry
